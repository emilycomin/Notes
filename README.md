# NOTAS 
Aplicação desenvolvida no evento NLW expert - trilha ReactJs da Rocketseat, consiste em adicionar notas em áudio ou texto. Utilizado React, TypeScript, Tailwind e a SpeechRecognition API.

## Executando 

```sh
npm install
npm run dev
```
